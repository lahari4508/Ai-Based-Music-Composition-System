# 🎶 AI-Based Music Composition System

## 📌 Overview

The **AI-Based Music Composition System** is an intelligent application that generates and plays music based on user mood.
It uses **Machine Learning + Natural Language Processing + Interactive UI** to convert human emotions into music genres and recommend songs accordingly.

---

## 🚀 Features

* 🎯 Mood detection from user text input
* 🎵 Genre prediction using ML model
* 🎧 Automatic music playback
* 📊 Animated audio waveform visualization
* ❤️ Add songs to playlist
* 📜 Track listening history
* ⬇ Download songs
* 🎨 Dynamic UI (colors change based on genre)

---

## 🧠 How It Works

1. User enters mood (text input)
2. System analyzes mood using NLP
3. Mood → mapped to music genre
4. Random song is selected from dataset
5. Music is played with animated UI
